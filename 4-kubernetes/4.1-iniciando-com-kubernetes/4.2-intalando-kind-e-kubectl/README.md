# Instalando Kind e Kubectl.

- Para a instalação do Kubectl e do Kind, efetuar o download no local `/usr/local/bin/` e adicionar o path ao `bashrc`.

```bash
sudo vi ~/.bashrc
```

```bash
export PATH=$PATH:/usr/local/bin/kind
```
